オリジナルのかなフォント「シンデレラ」と、それに合うようにM+FONTSを加工した、M+FONTS派生フォントです。

飴鞭シンデレラの後継フォントとして作成しました。
後継フォントの作成については、飴鞭ゴシック作者のtakumiさんに許可を頂いております。

## 仕様・収録文字について

−ひらがな・カタカナ・漢字。
